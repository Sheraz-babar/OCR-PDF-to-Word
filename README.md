# OCR-PDF-to-Word

A Python-based OCR system that converts scanned PDF documents into editable Word files.

## Features
- Converts multi-page PDFs to editable Word documents.
- Preprocesses images for better OCR accuracy (grayscale + thresholding).
- Handles text extraction and preserves page-wise formatting.
- Optimized for multi-page processing with execution time tracking.

## Technologies Used
Python, pytesseract, pdf2image, PIL (Pillow), python-docx, Image Preprocessing, PDF Handling, Text Extraction
